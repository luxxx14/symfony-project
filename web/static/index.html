Eng

Открытая платформа
для разработки бизнес-приложений с веб-интерфейсом

Разработав бизнес-приложение, запускайте его на той СУБД, которую предпочитает ваш заказчик.
Визуально проектируйте таблицы базы данных на языке CelestaSQL.
Быстро создавайте экранные формы, связанные с данными таблицы.
Выводите отчёты в форматах Excel, PDF и напрямую на принтер

Компоненты платформы

Showcase

Система разработки кросс-платформенных веб-решений в условиях постоянно меняющихся требований бизнеса

Celesta

Платформа для быстрой разработки бизнес‑приложений на языке Python, работающих на реляционных базах данных

Lyra

Средство быстрой разработки форм пользовательского интерфейса, привязанных к курсорам Celesta

Flute

Система управления фоновым запуском в нескольких параллельно выполняемых процессах Celesta-скриптов

Mellophone

Компонент платформы КУРС, специализированно решающий задачи аутентификации

Theremin

Система вычисления расчётных показателей в аналитическом хранилище многомерных данных

Xylophone

Система формирования сложноструктурированных отчётов
