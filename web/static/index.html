Открытая платформа
для разработки бизнес-приложений с веб-интерфейсом

Разработав бизнес-приложение, запускайте его на той СУБД, которую предпочитает ваш заказчик.
Визуально проектируйте таблицы базы данных на языке CelestaSQL.
Быстро создавайте экранные формы, связанные с данными таблицы.
Выводите отчёты в форматах Excel, PDF и напрямую на принтер

Компоненты платформы

Showcase

Конструирование веб-интерфейса бизнес‑систем, работающего в Java Servlet Container

Celesta

Разработка приложений в условиях меняющихся требований бизнеса на языке Jython с использованием реляционной базы данных

Flute

REST-сервисы, обработчики очередей и периодические задания на базе Celesta-процедур

Mellophone

Модуль аутентификации пользователей

Xylophone

Формирование печатных форм и документов

Основные преимущества

CelestaSQL: код бизнес-логики не зависит от типа базы данных

Мы поддерживаем SQL Server, Oracle, PostgreSQL и H2

Автоматическое обновление структуры базы данных через идемпотентный DDL

CelestaUnit: быстрое и лёгкое модульное тестирование процедур обработки данных

Сервис-ориентированная архитектура
